  When you are using PySerial, it may cause a delay in the system (mine was 10 second), especially you are sending packages continuously. Use the following four steps to avoid this issue: 1. Send the package only when the value has changed. 2. Set a time_out for your program 3. Set a delay in your loop of Arduino, Raspberry Pi... 4. Setting try and except format to the data to avoid any error and do a ser.flush() to clean up any bad data.
